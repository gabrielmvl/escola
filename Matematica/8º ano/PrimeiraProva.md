# Números naturais e inteiros
## Conjunto dos números naturais

- Números positivos!

Números naturais:
- <span style="font-size: 20px;">__$|N = \{0, 1, 2, 3, 4, 5...\}$__</span>

Números naturais não nulos:
- <span style="font-size: 20px;">__$|N^* = \{ 1, 2, 3, 4, 5...\}$__</span>

## Conjunto dos números inteiros

- Números positivos e negativos!

Números inteiros:

- <span style="font-size: 20px;">__$Ƶ = \{...-2, -1, 0, 1, 2...\}$__</span>

Números inteiros não nulos:

- <span style="font-size: 20px;">__$Ƶ^* = \{...-2, -1, 1, 2...\}$__</span>

## Regra de sinais

### Adição e subtração

- Sinais iguais --> Conserva-se o sinal e soma.

    - <span style="font-size: 20px;">$30+37=67$</span>

- Sinais diferentes --> Conserva-se o sinal do maior e subtrai.
    - <span style="font-size: 20px;">$178-111=67$</span>

### Multiplicação e divisão

- Sinais iguais --> <span style="font-size: 20px;">__$+$__</span>

    - <span style="font-size: 20px;">__$(+)\cdot(+)=(+)$__</span>
    - <span style="font-size: 20px;">__$(-)\cdot(-)=(+)$__</span>

- Sinais diferentes --> <span style="font-size: 20px;">__$-$__</span>

    - <span style="font-size: 20px;">__$(-)\cdot(+)=(-)$__</span>
    - <span style="font-size: 20px;">__$(+)\cdot(-)=(-)$__</span>

# Potenciação

- A potenciação é a multiplicação de fatores iguais.

    - <span style="font-size: 20px;">__$X⁴ = X \cdot X \cdot X \cdot X$__</span>
    ­
    - <span style="font-size: 20px;">__$3³ = 3 \cdot 3 \cdot 3 = 27$__</span>

## Base negativa

- Se expoente for ímpar o resultado da potencia será negativo.

    - <span style="font-size: 20px;">__$(-3)³=-27$__</span>

- Se expoente for par o resultado da potencia será positivo.

    - <span style="font-size: 20px;">__$(-4)²=16$__</span>

    - Exceto, se não houver parênteses:

        - <span style="font-size: 20px;">__$-4²=-16$__</span>

## Expoente negativo

- Deve-se inverter a base.

    - <span style="font-size: 20px;">__$(\frac{X}{Y})^{-n} = (\frac{Y}{X})^{n}$__</span>

    ­

    - <span style="font-size: 20px;">__$(\frac{5}{4})^{-2} = (\frac{4}{5})^{2} = \frac{16}{25}$__</span>

## Propriedades da potenciação

- Todo número elevado a 1 dá ele mesmo.

    - <span style="font-size: 20px;">__$X¹=X$__</span>
    - <span style="font-size: 20px;">__$6¹=6$__</span>

- Todo número elevado a 0 resulta em 1.

    - <span style="font-size: 20px;">__$X⁰=1$__</span>
    - <span style="font-size: 20px;">__$7⁰=1$__</span>

- Na multiplicação de potencias de mesma base soma-se os expoentes

    - <span style="font-size: 20px;">__$X⁵\cdot X²=X^{5+2}=X⁷$__</span>
    - <span style="font-size: 20px;">__$5³\cdot 5=5^{3+1}=5⁴=625$__</span>

- Na divisão de potencias de mesma base soma-se os expoentes

    - <span style="font-size: 20px;">__$X⁵\cdot X²=X^{5+2}=X⁷$__</span>
    - <span style="font-size: 20px;">__$5³\cdot 5=5^{3+1}=5⁴=625$__</span>

- Quando se há uma potencia de uma outra potencia multiplica-se os expoentes

    - <span style="font-size: 20px;">__$(X³)⁵ = X^{3 \cdot 5}= X¹⁵$__</span>
    - <span style="font-size: 20px;">__$(2⁴)³ = 2^{4 \cdot 3}= 2¹²=4096$__</span>

## Potenciação com radicação com expoente fracionário

<span style="font-size: 20px;">__$X^{\frac{3}{4}}=\sqrt[4]{X^{3}}$__</span>

<span style="font-size: 20px;">__$8^{\frac{2}{3}}=\sqrt[3]{8^{2}}=\sqrt[3]{64}=4$__</span>

<span style="font-size: 20px;">__$\sqrt[8]{X³²}=X^{32/8}= X^{4}$__</span>

<span style="font-size: 20px;">__$\sqrt[5]{5²⁵}=5^{25/5}= 5^{5}=3125$__</span>
